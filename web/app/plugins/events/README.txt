A powerful plugin to manage events with custom fields for enhanced flexibility and organization.

== Description ==

Events Manager is a comprehensive plugin for creating and managing events in WordPress. With custom fields support, you can add detailed information about your events such as location, date, time, organizer, and more. 

### Features:
- Add, edit, and delete events.
- Use custom fields to add event-specific information.
- Display events in a customizable calendar view.
- Shortcodes for embedding event details anywhere on your site.
- Integrates seamlessly with your WordPress theme.

This plugin is ideal for organizations, event planners, or anyone who wants a flexible event management solution.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `events` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under the "Events" menu in the WordPress dashboard.
4. Use the shortcode `[events_list]` to display a list of events or `[event_calendar]` to display a calendar.

== Frequently Asked Questions ==

= How do I add an event? =
Go to the "Events" menu in your WordPress dashboard, click "Add New," and fill in the event details using the custom fields provided.

= Can I customize the look of the events list or calendar? =
Yes, the plugin provides basic styling options. For advanced customization, you can override the templates in your theme.

= Does the plugin support recurring events? =
Currently, recurring events are not supported but may be included in future updates.
